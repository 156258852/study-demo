属性 包含内容 说明
clientWidth 内容宽度 + padding 不包括边框、外边距和滚动条
clientHeight 内容高度 + padding 不包括边框、外边距和滚动条
offsetWidth 内容 + padding + border 包括边框，但不包括外边距和滚动条
offsetHeight 内容 + padding + border 包括边框，但不包括外边距和滚动条
scrollWidth 完整内容宽度 包括因溢出而被隐藏的部分
scrollHeight 完整内容高度 包括因溢出而被隐藏的部分

属性解释

  • clientWidth/clientHeight: 元素内部的尺寸，包括内容和内边距(padding)，但不包括边框(border)、外边距(margin)或滚动条
  • 计算公式:
    • clientWidth = CSS width + CSS padding-left + CSS padding-right
    • clientHeight = CSS height + CSS padding-top + CSS padding-bottom
  • 对于本例中的元素（宽300px，高200px，padding 20px，border 10px）：
    • clientWidth = 300 + 20 + 20 = 340px
    • clientHeight = 200 + 20 + 20 = 240px
    • offsetWidth = 300 + 20 + 20 + 10 + 10 = 360px
    • offsetHeight = 200 + 20 + 20 + 10 + 10 = 260px

演示元素结构

下面的彩色框是一个具有以下CSS属性的元素：

  • width: 300px
  • height: 200px
  • padding: 20px
  • border: 10px solid
  • background-color: #ecf0f1
  • overflow: auto

蓝色边框表示元素的边界，内部的紫色渐变区域是内容区域。

实际应用场景

  • 响应式设计中计算可用空间
  • 动态调整元素大小
  • 实现自定义滚动条
  • 创建全屏或适应性布局
  • 计算元素相对于视口的位置
