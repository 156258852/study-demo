选择器语法说明

:is() 语法：

:is(selector1, selector2, ...) {
    /* 样式规则 */
}

:where() 语法：

:where(selector1, selector2, ...) {
    /* 样式规则 */
}

主要区别：

  • :is() - 采用选择器列表中最高的特异性
  • :where() - 特异性始终为0

浏览器支持说明

:is() 和 :where() 是较新的CSS特性，可能不被所有浏览器支持。在生产环境中使用时，请检查目标浏览器的兼容性。

  • Firefox: 支持
  • Chrome: 支持
  • Safari: 支持
  • Edge: 支持
  • IE: 不支持
