三栏布局示例

展示三种不同的三栏布局实现方式：Flexbox、Grid 和 浮动

1. Flexbox 布局

左侧边栏

这是左侧边栏的内容。在 Flexbox 布局中，我们使用 flex 属性来控制各列的宽度比例。

  • Flexbox 是一维布局模型
  • 非常适合响应式设计
  • 可以轻松实现垂直居中

主要内容区域

这是主要内容区域。Flexbox 布局的优势在于它能轻松处理不同屏幕尺寸的适配问题。

Flexbox 布局的核心概念是容器和项目。容器通过 display: flex 声明，项目则自动成为容器的子元素。

主要属性包括：flex-direction、flex-wrap、justify-content、align-items 等。

右侧边栏

这是右侧边栏的内容。在 Flexbox 中，我们可以使用 flex: 1、flex: 2 等来设置比例。

  • flex: 1 表示占据剩余空间的1份
  • flex: 2 表示占据剩余空间的2份
  • 可以轻松调整各列的比例

2. Grid 布局

左侧边栏

这是左侧边栏的内容。在 Grid 布局中，我们使用 grid-template-columns 来定义列宽。

  • Grid 是二维布局模型
  • 可以同时控制行和列
  • 更精确的网格控制

主要内容区域

这是主要内容区域。Grid 布局通过 grid-template-columns: 1fr 2fr 1fr 定义三列的比例。

CSS Grid 布局是目前最强大的 CSS 布局系统之一，它是一个基于网格的二维布局系统。

Grid 布局非常适合复杂的网页布局，可以轻松实现各种对齐和分布方式。

右侧边栏

这是右侧边栏的内容。Grid 布局中使用 fr 单位来表示可用空间的分数。

  • 1fr 表示可用空间的1份
  • 2fr 表示可用空间的2份
  • 可以创建复杂的网格结构

3. 浮动 布局

左侧边栏

这是左侧边栏的内容。在浮动布局中，我们使用 float 属性来实现多列布局。

  • 浮动最初用于文字环绕效果
  • 后来被用于页面布局
  • 需要清除浮动来避免布局问题

主要内容区域

这是主要内容区域。浮动布局通过 float: left 和 float: right 实现多列布局。

浮动布局是较早的 CSS 布局方式，在 Flexbox 和 Grid 出现之前广泛使用。

需要注意清除浮动以避免父元素高度塌陷的问题，通常使用 overflow: hidden 或 clearfix 技术。

右侧边栏

这是右侧边栏的内容。在浮动布局中，宽度通常使用百分比或固定值。

  • 需要计算各列的宽度和间距
  • 容易出现高度塌陷问题
  • 响应式处理相对复杂
