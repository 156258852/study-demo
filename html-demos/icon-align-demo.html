📝 总结：核心要点

图标与第一行对齐的通用公式：

外层容器: align-items: flex-start（顶部对齐）
图标容器: height = 行高 + 内部 flex 居中
  • 关键1：外层用 flex-start 让图标容器和文本容器顶部对齐
  • 关键2：图标容器高度 = 文本行高（等于第一行的"框"高度）
  • 关键3：图标容器内部用 flex center 让图标在"行高框"内居中
  • 简化版：图标直接设置 line-height + vertical-align: middle（方案2）
