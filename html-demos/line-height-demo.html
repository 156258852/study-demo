💡 关键知识点

核心概念

  • 行高 (line-height)：一行文字所占据的垂直高度
  • 字体大小 (font-size)：文字本身的视觉高度
  • 行间距 (leading)：行高 - 字体大小，即文字上下的空白空间
  • 半行距：行间距的一半，均匀分布在文字上下

最佳实践

  • 推荐使用无单位数字：如 line-height: 1.5，子元素继承比例值
  • 正文文本：行高建议 1.5 - 1.8 之间
  • 标题文本：行高建议 1.2 - 1.4 之间
  • 单行垂直居中：设置 height = line-height
  • 多行垂直居中：使用 flexbox 或 grid 布局
